# ไซต์แมพ — ระบบจัดการแข่งเกม

ระบบเป็น SPA (React Router) แบ่งการเข้าถึงเป็น 3 ระดับ: **ผู้ที่ยังไม่ล็อกอิน (Guest)**, **ผู้ใช้ที่ล็อกอินแล้ว (Player/Admin ทั่วไป)**, และ **เฉพาะแอดมิน**

```
/
├── /login                          [Guest] เข้าสู่ระบบ (อีเมล/รหัสผ่าน + Google)
├── /signup                         [Guest] สมัครสมาชิก
│
├── /complete-profile               [ล็อกอินแล้ว] กรอกข้อมูลเพิ่มหลังสมัครด้วย Google
│
├── /                               [ล็อกอินแล้ว] หน้าแรก — แยกเนื้อหาตาม role
│   ├─ (player) → กล่อง "ต้องทำต่อ" + "ลีคของฉัน"
│   └─ (admin)  → แผงแอดมิน (แดชบอร์ดสรุป)
│
├── /profile                        [ล็อกอินแล้ว] แก้ไขโปรไฟล์ตัวเอง
│
├── /leagues                        [ล็อกอินแล้ว] หน้ารวมลีคสาธารณะ (แท็บ เปิดรับสมัคร/กำลังแข่ง/จบแล้ว)
│   └── /leagues/:leagueId          [ล็อกอินแล้ว] รายละเอียดลีค + ตารางคะแนน/สายถ้วย
│
├── /matches/:matchId               [ล็อกอินแล้ว] หน้าแมตช์ — ส่งผล/ดูผล/ติดต่อคู่แข่ง
│
└── /admin/...                      [เฉพาะแอดมิน]
    ├── /admin/leagues              รายการลีคทั้งหมด (มุมมองแอดมิน)
    │   ├── /admin/leagues/new      สร้างลีคใหม่
    │   └── /admin/leagues/:leagueId  จัดการลีค (เริ่ม/เตะ/แพ้บาย/ย้อนรอบ/ลบ)
    ├── /admin/disputes             คิวข้อพิพาท + คำขอติดต่อไม่ได้ (ถ้วย) + ค้นหาด้วยรหัสแมตช์
    └── /admin/stale-matches        รายการแมตช์ค้าง (ข้อมูลประกอบการตัดสินใจ)

* (เส้นทางอื่นที่ไม่ตรงกับรายการด้านบน) → หน้า 404
```

## หมายเหตุการเข้าถึง
- ทุกหน้ายกเว้น `/login` และ `/signup` ต้องล็อกอินก่อน (เด้งไป `/login` อัตโนมัติถ้ายังไม่ล็อกอิน)
- ผู้ใช้ที่ล็อกอินแล้วแต่ยังไม่มีโปรไฟล์ (เพิ่งสมัครด้วย Google ครั้งแรก) จะถูกบังคับไปที่ `/complete-profile` ก่อนใช้งานหน้าอื่นได้
- เส้นทางในกลุ่ม `/admin/...` เข้าได้เฉพาะผู้ใช้ที่มี `role: admin` เท่านั้น (มีคนเดียวในระบบ) ผู้เล่นทั่วไปที่พิมพ์ URL ตรง ๆ จะถูกเด้งกลับหน้าแรก
- ไม่มีหน้าใดเปิดให้ search engine เข้าถึงได้แบบสาธารณะ เพราะทุกข้อมูลอยู่หลังระบบล็อกอิน
